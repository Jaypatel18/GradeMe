About Us




GradeMe is a website that is meant to help you keep track of your academics as well as enhance how you approach each class. With GradeMe you can strategically plan what grade you want to obtain in each class. GradeMe also provides tools to help you achieve these grades like user uploaded study guides, notes, previous exams, etc. You can also ask classmates for help with the chat feature. We hope that with GradeMe you will be able to achieve you academic goals as well as help others achieve theirs.
